# Simulador de Impostos de Loja em Python

Projeto simples desenvolvido para praticar lógica de programação com Python, simulando regras de imposto aplicadas a produtos de uma loja.

## 🔧 Funcionalidades

- Cálculo de imposto com base no valor do produto:
  - Até R$1000 → 10%
  - Acima de R$2000 → 15%
  - Acima de R$15000 → 20%
- Identificação de produtos premium
- Contagem de produtos premium
- Soma total de impostos arrecadados
- Cálculo do faturamento total

## 💡 Conceitos aplicados

- Estruturas condicionais (if, elif, else)
- Laços de repetição (for)
- Acumuladores
- Manipulação de listas
- Lógica de negócio simples
